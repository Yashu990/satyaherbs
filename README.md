# Satyaherbs — Premium Ayurvedic E-commerce

A modern, high-performance e-commerce frontend built for **SATYAHERBS**, featuring a premium aesthetic inspired by luxury ayurvedic brands like Kapiva.

![SATYAHERBS Preview](https://via.placeholder.com/1200x600/1a0f00/c8a45a?text=SATYAHERBS+PREMIUM+AYURVEDA)

## ✨ Features

- **🛍️ Complete Shopping Journey**: End-to-end frontend flow from discovery to order success.
- **🛒 Global Cart Management**: Persistent cart state with dynamic badge updates.
- **💳 Multi-step Checkout**: Professional shipping and payment interface with secure mock validation.
- **🌿 14 Specialized Categories**: Dedicated pages for everything from Men's & Women's Health to Pure Himalayan Shilajit.
- **📱 Fully Responsive**: Optimized for desktop, tablet, and mobile with smooth CSS transitions.
- **⚡ Built with Vite**: Lightning-fast development and optimized production builds.
- **✨ Premium UI/UX**: Dark mode accents, gold gradients, and botanical motifs for a high-end natural feel.

## 🛠️ Tech Stack

- **React 18** (Functional Components & Hooks)
- **Vite** (Build Tool)
- **Tailwind CSS** (Styling)
- **React Router 6** (Navigation)
- **Context API** (State Management)

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/Yashu990/satyaherbs.git
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run in development mode**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```

## 📦 Project Structure

- `/src/components`: Reusable UI components (Header, Footer, CartSidebar, etc.)
- `/src/pages`: Individual category and functional pages (DailyWellness, Checkout, etc.)
- `/src/context`: Global state management (CartContext)
- `/src/images`: Optimized botanical and product assets
